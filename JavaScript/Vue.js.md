# Vue.js
  ## Vue.js란?
  프론트엔드 프레임워크. 
  
  ### 선언적 렌더링
  예제
  ```
  <div id="app">
  {{ message }}
</div>
  ```
  
  ```
  var app = new Vue({
  el: '#app',
  data: {
    message: '안녕하세요 Vue!'
  }
})
```
