# EL과 JSTL
  ## 1. EL(Expression Language)
  ### 1-1. EL이란
  JSP의 출력 언어를 대체하는 표현 언어
  
   <%= abc %>를 ${abc}로 간단하게 사용할 수 있게 하였고, JSTL의 Core에서 c를 이용해 <%= if%>문을 <c:if>, <%=for%>문을 <c:forEach>로 대체하여 사용합니다.
